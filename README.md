# Вектор Экспертиза — лендинг (HTML/CSS/JS)

## Как открыть
- Откройте файл `index.html` двойным кликом в браузере.

## Что уже сделано
- Адаптивная мобильная версия (mobile-first) + сетки для планшета/десктопа
- 2 шрифта: **Manrope** (текст) и **PT Serif** (заголовки)
- Цветовая система 3–4 цвета: **primary blue** (CTA), мягкий **accent** (дозировано), нейтральные тёмные поверхности
- Рабочие кнопки CTA: открывают модальное окно / ведут по якорям
- Форма обратной связи в модальном окне (Имя, Телефон, Email, Сообщение) — отправка на **vector-kzn@mail.ru** через [Formsubmit.co](https://formsubmit.co)
- После отправки — переход на страницу **Спасибо** (`thank-you.html`)

## Форма заявок
Заявки отправляются на **vector-kzn@mail.ru**. При первой отправке Formsubmit пришлёт на эту почту письмо с подтверждением — нужно перейти по ссылке, чтобы активировать приём заявок. Сменить email можно в атрибуте `action` формы в `index.html` и `вектор/index.html` (формат: `https://formsubmit.co/ajax/ваш@email.ru`).

## Где менять контакты
В файле `index.html`:
- Телефон: блок `Телефон` и атрибут `href="tel:..."`
- Email: блок `Email` и `href="mailto:..."`

## Где смотреть “заявки”
- `index.html` — главная; форма по кнопкам «Заказать» / «Купить».
- `thank-you.html` — страница «Спасибо» после отправки.
